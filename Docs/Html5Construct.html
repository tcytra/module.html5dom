Html5Construct

Description

The Html5Construct object will evaluate a provided HTML element constructor in string format and retrieve the parameters as instance properties.

Properties

public string class
The requested class name(s) in the constructor; Default value: null
public string id
The requested element id in the constructor; Default value: null
public string name
The requested element node name in the constructor; Default value: div

Methods

public __construct( )
Create an instance of the Html5Construct
public bool able( )
Indicates whether a provided constructor is a viable HTML5 element

Example Use

  1. <?php
  2. // Set an instance of Html5Construct with a constructor argument
  3. $construct = Html5Construct::Set("div#interface.wrapper");
  5. // Dump the Html5Construct
  6. var_dump($construct);
  8. object(Html5Construct)#1 (3) {
  9. ["id"]=>
  10. string(9) "interface"
  11. ["class"]=>
  12. string(7) "wrapper"
  13. ["name"]=>
  14. string(3) "div"
  15. }
