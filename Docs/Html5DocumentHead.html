Class Html5DocumentHead Extends Html5

Description

The Html5DocumentHead object provides methods for adding and manipulating the various nodes in the DOMDocument <head> element.

Properties

protected object $objnode
The local instance target PHP DOMElement
public string $objtype
The type-name of this object; Value: documenthead

Methods

public __construct( [array $config = null] )
Create an instance of the Html5DocumentHead and process the provided $config
public void create( )
Create the document <head> tag and append it to the parent target
public object favicon( [string $href = "/favicon.png"] )
Add a document <link> tag and attributes for the icon to the node tree
Returns $this instance of the Html5DocumentHead
public object javascript( string $src[, string $code = null] )
Add a document <script> tag and attributes to the html5 node tree
public object meta( array $attr )
Add a document <meta> tag and attributes to the Html5 node tree
public object stylesheet( string $href )
Add a document <link> tag and attributes to the html5 node tree
public object title( string $text[, int $amend = 0, string $join = null] )
Add the document <title> tag to the html5 node tree

Example Use

  1. <?php
  2. // Create an instance of the Html5Document object
  3. $html5 = new Html5Document();
  5. // Append the various <head> elements to the $html
  6. $html5->head->meta(["viewport"=>"width=device-width,initial-scale=1"])
  7. ->stylesheet("/css/styles.css")
  8. ->javascript("/js/script.js")
  9. ->javascript(null, "(function(){ /* inline script */ })();")
  10. ->title("Class Html5DocumentHead Extends Html5")
  11. ->favicon("/favicon.png");
  13. // Output the document html
  14. $html5->save()->write();
  16. // Result:
  17. <!DOCTYPE html>
  18. <html lang="en">
  19. <head>
  20. <meta charset="utf-8">
  21. <meta viewport="width=device-width,initial-scale=1">
  22. <title>Class Html5DocumentHead Extends Html5</title>
  23. <link rel="icon" href="/favicon.png">
  24. <link rel="stylesheet" type="text/css" href="/css/styles.css">
  25. <script type="text/javascript" src="/js/script.js"></script><script type="text/javascript">
  26. (function(){ /* inline javascript */ })();
  27. </script>
  28. </head>
  29. <body></body>
  30. </html>
